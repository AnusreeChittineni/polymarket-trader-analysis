# CSE 6242 Polymarket Trader Analysis

DESCRIPTION:

Our code package includes our processing of the raw datasets through duckdb, our clustering experimentations, and the visualization platform code. 

INSTALLATION:

- go to https://github.com/Jon-Becker/prediction-market-analysis?tab=readme-ov-file and download the data.tar.zst from the readme
- run create_subset/sample.ipynb
- run create_subset/analysis.ipynb
- run explore_data/append_categories.py on trader_stats.csv generated from analysis.ipynb
- run dbscan.py which takes in samples/updated_trader_stats.csv

EXECUTION:

- run frontend
